# YOGA LIFE VN

  A full-stack multi-center yoga management web app built with React, Express, TypeScript, and PostgreSQL (Supabase).

  ## Features
  - Multi-center management (Hà Nội, TP.HCM, Đà Nẵng)
  - 3 roles: Super Admin, Center Admin, Student
  - Full CMS for website content (hero, teachers, studios, videos, reviews)
  - Class Library & Schedule management
  - Membership & enrollment system
  - EN/VI language support
  - Connected to Supabase PostgreSQL

  ## Tech Stack
  - **Frontend:** React + Vite + TypeScript + Tailwind CSS
  - **Backend:** Express.js + TypeScript
  - **Database:** PostgreSQL (Supabase) via Drizzle ORM
  - **Monorepo:** pnpm workspaces
